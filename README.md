## О проекте
Данный проект является учебным для меня.
В данном плагине вы сможете:
* [X] [Добавляет кастомизированный Tab](#кастомизированный-tab)
    - [ ] файлы конфигурации
    - [ ] команды для таба
* [X] [Добавляет кастомизированный чат](#кастомизированный-чат)
  - [ ] файлы конфигурации
  - [ ] команды редактирования текста
* [ ] [Добавляет кастомизированный Sidebar](#кастомизированый-sidebar)
    - [ ] файлы конфигурации
    - [ ] команды для текста
* [ ] [Добавляет текст под никами игроков](#текст-под-никами)
    - [ ] файлы конфигурации
    - [ ] команды для текста
* [X] [Добавляет свои кастомные значки](#значки-в-чате)

> [!IMPORTANT]
> Для значков в чате нужно скачать ресурспак, который находится в корневой папке плагина.

Если плагин станет популярным, я добавлю возможность редактировать настройки через команды в игре, без необходимости редактировать файлы плагина.
